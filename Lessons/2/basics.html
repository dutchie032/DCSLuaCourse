Basics

We'll start with some basics so you can get started.

How to write to the console

In this part of the course we'll go over printing messages.
As is commong practice we do this and begin with "Hello world".
We do this by using

print("Hello World")


OR dependent on settings

io.write("Hello World")


for this course I would say just use the first of the two.

In DCS however it works a little different. There's no "Console" visible in DCS, only a log file.
We'll go over how to log text a little later.

Variables

Variables are a big core concept to almost any coding language. It's a way to store some data and retrieve it later.
We'll not go into the details where that data is stored (heap, stack, etc.), since generally this doesn't matter when writing LUA.
You can declare a value. When doing so you tell the program you want it to keep track of some data for you.
For example:

local a

Now we have a variable "a" which is now declared and local to this "scope".

If we now want to store something in that variable we can "assign" a value to that variable.
For example:

a = 1


You can also instantiate the variable when creating it.
For Example:

local b = 2


You can pretty much store anything in a variable which can help write your code and make it easier to read.

Combining strings

A string is a bit of text. This can again be stored in a variable. For instance:

local a = "something" \r\n local b = "else" \r\n local c = a .. " " .. b -- takes a and b and combines them into c \r\n print(c) \r\n -- Output: "something else"
